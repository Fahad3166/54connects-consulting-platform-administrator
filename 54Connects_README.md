# 54Connects — Platform Administration & UX Consulting

**Client:** 54Connects (US-based African media platform)  
**Role:** Remote Platform Administrator & UX Consultant  
**Duration:** 2025 – Present  
**Website:** [54connects.com](https://54connects.com)

---

## About 54Connects

54Connects is a US-based African media company with studios for rent and a digital platform promoting African content and culture. The platform serves multiple user types including reporters across Africa who upload news stories and videos, subscribers who access on-demand content, restaurant owners, and platform administrators. Think of it as a combination of a media newsroom, a Netflix-style on-demand service, and a marketplace — all running on one backend.

---

## My Role

I work remotely as a platform administrator and UX consultant, sitting between the client, the development team, and the design team. My job is to understand what the platform needs to do, identify what is not working, and translate that into clear guidance for the people building it.

This covers three main areas:

**Platform Administration**
- Managing user roles and permissions across account types: reporters, whistleblowers, I-Witness contributors, subscribers, and administrators
- Reviewing platform functionality before and after updates
- Producing pre-launch review reports identifying bugs, broken links, missing content, and UX issues

**Dashboard Redesign & Feature Consulting**
- Audited the existing backend dashboard and identified critical usability problems — irrelevant metrics, disorganized sidebar, no real-time media monitoring
- Produced a full Backend Dashboard Redesign & Feature Enhancement Report covering navigation architecture, dashboard layout, real-time media analytics, AI-powered insights, moderation workflows, and e-commerce stats
- Proposed and documented new features including live TV and radio viewer monitoring, I-Witness auto-approval queue, AI content suggestions, and a media calendar view

**Pre-Launch QA & Content Review**
- Reviewed the full platform after each development update
- Identified and documented issues: dummy content still showing, broken links with no fallback, missing social media connections, test posts not removed before go-live
- Recommended technical improvements such as connecting the YouTube vlog page via YouTube API for automatic video syncing rather than manual uploads, reducing server load and improving page speed
